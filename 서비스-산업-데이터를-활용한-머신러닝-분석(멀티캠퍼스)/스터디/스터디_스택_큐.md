7차시 4일차 - 괄호검사

오류 10개 중 3개 맞음. 왜??????

```python
T = int(input())

for t in range(T):

    checkList = []
    
    myStr = input()  

    for i in range(len(myStr)):
        if myStr[i] == '(':
            checkList.append(myStr[i])
        if myStr[i] == ')':
            checkList.pop()
        
    if len(checkList) != 0:
        print(f"#{t+1} 0")
        break
        
    else:
        if myStr[i] == '}':
            checkList.pop()
        if myStr[i] == '{':
            checkList.append(myStr[i])
    if len(checkList) != 0:
        print(f"#{t+1} 0")
    else:
        print(f"#{t+1} 1")
```
