1. INNER JOIN

   - CARTESIAN PRODUCT에서 중복 제거. 양 쪽 테이블에 모두 데이터가 존재해야 함.
   - 해당 조건에 맞는 열들만 추출.

   ```SQL
   SELECT *
   FROM CUSTOMERS
   JOIN ORDERS ON CUSTOMERS.CUSTOMERID = ORDERS.CUSTOMERID
   ```

   

   * EX1_런던에 위치한 공급자가 생산한 상품 목록 조회

   ```SQL
   SELECT *
   FROM SUPPLIERS AS S
   JOIN PRODUCTS AS P ON S.SUPPLIERID = P.SUPPLIERID
   ```
